# Process Manufacturing

Continuous and batch production processes.

## Scope

This subsector covers manufacturing of formulated products, including:

- Batch processing and recipes
- Quality control and testing
- Process parameter monitoring
- Lot tracking and traceability
- Regulatory compliance (FDA, GMP)

## Models

| Model | Description |
|-------|-------------|
| `batch-processing/` | Recipe execution and batch records |
| `quality-control/` | Sampling, testing, and release |
