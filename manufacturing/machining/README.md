# Machining

Precision manufacturing and CNC operations.

## Scope

This subsector covers machine shop operations, including:

- CNC programming and execution
- Tool management and wear tracking
- Precision measurement and QC
- Machine utilization and OEE
- Job shop scheduling

## Models

| Model | Description |
|-------|-------------|
| `cnc-operations/` | Program management and machine control |
| `precision-manufacturing/` | Tolerances, measurements, and calibration |
