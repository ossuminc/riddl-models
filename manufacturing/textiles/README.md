# Textiles

Fabric production and apparel manufacturing.

## Scope

This subsector covers textile and apparel manufacturing, including:

- Fabric production and finishing
- Pattern making and grading
- Cut, make, and trim operations
- Dyeing and printing
- Garment assembly

## Models

| Model | Description |
|-------|-------------|
| `fabric-production/` | Weaving, knitting, and finishing |
| `apparel-manufacturing/` | Pattern, cut, sew, and assembly |
