# Discrete Manufacturing

Production of distinct, countable items.

## Scope

This subsector covers manufacturing of discrete products, including:

- Bill of materials management
- Work order creation and tracking
- Assembly line operations
- Quality inspections
- Production scheduling

## Models

| Model | Description |
|-------|-------------|
| `bill-of-materials/` | BOM structure, revisions, and engineering changes |
| `work-order-management/` | Production orders and shop floor execution |
| `assembly-operations/` | Assembly line sequencing and tracking |

## Related Patterns

- `patterns/entity/aggregate-root/` - For BOM hierarchies
- `patterns/workflow/process-manager/` - For production workflows
