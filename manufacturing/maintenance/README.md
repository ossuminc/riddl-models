# Maintenance

Equipment maintenance and asset lifecycle.

## Scope

This subsector covers industrial maintenance operations, including:

- Preventive maintenance scheduling
- Work order management
- Spare parts inventory
- Asset lifecycle tracking
- Predictive maintenance

## Models

| Model | Description |
|-------|-------------|
| `equipment-maintenance/` | PM schedules, work orders, and technicians |
| `asset-lifecycle/` | Asset registration, depreciation, and disposal |

- `patterns/entity/event-sourced/` - For maintenance history
