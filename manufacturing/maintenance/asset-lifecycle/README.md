# Asset Lifecycle

Equipment acquisition, depreciation, and disposal.

## Scope

This model covers asset lifecycle management, including:

- Asset acquisition and commissioning
- Depreciation calculation
- Capital vs expense decisions
- Asset transfers and relocations
- Retirement and disposal

## Key Concepts

| Concept | Description |
|---------|-------------|
| Asset | A capital equipment item |
| Acquisition | Purchase and installation of an asset |
| Depreciation | Value reduction over time |
| Disposal | Retirement or sale of an asset |
