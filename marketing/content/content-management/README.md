# Content Management

Marketing content creation, editorial review, and publishing workflow.

## NAICS Code

**541810** - Advertising Agencies

## Scope

This model covers content management operations, including:

- Content creation from briefs with writer assignment
- Draft submission and media attachment
- Editorial review with comments and revision cycles
- Content approval and version tracking
- Scheduled and immediate publishing
- SEO metadata management
- Content performance metrics (views, shares, conversions)

## Key Concepts

| Concept | Description |
|---------|-------------|
| Content | A marketing content item (blog, article, etc.) |
| ContentBrief | Assignment with objective, audience, and deadline |
| ReviewComment | Editor feedback on content drafts |
| ContentMetrics | Views, bounce rate, shares, and conversions |
