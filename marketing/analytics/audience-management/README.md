# Audience Management

Customer segments, targeting, and activation.

## NAICS Code

**541910** - Marketing Research and Public Opinion Polling

## Scope

This model covers audience management, including:

- Segment definition and criteria
- Audience building and sizing
- Profile enrichment
- Platform activation (ad platforms, email)
- Lookalike and suppression audiences

## Key Concepts

| Concept | Description |
|---------|-------------|
| Segment | A defined group of customers |
| Audience | An activated segment for targeting |
| Profile | A customer's attributes and behaviors |
| Activation | Sending audiences to platforms |
