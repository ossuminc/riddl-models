# Ad Operations

Creative management, targeting, and campaign reporting.

## Scope

This model covers ad operations, including:

- Creative asset management
- Targeting configuration
- Campaign trafficking and scheduling
- Performance monitoring
- Reporting and analytics

## Key Concepts

| Concept | Description |
|---------|-------------|
| Creative | An advertising asset (image, video, copy) |
| Campaign | A set of ads with targeting and budget |
| Placement | Where and when an ad appears |
| Report | Aggregated performance metrics |
