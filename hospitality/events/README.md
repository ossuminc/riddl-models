# Events

Event planning, venue management, and registration.

## Scope

This subsector covers event and conference management, including:

- Event registration and ticketing
- Venue booking and layout
- Speaker and session management
- Attendee engagement
- On-site operations

## Models

| Model | Description |
|-------|-------------|
| `event-registration/` | Attendee signup, payments, and badges |
| `venue-management/` | Space booking, setup, and logistics |

## Related Patterns

- `patterns/saga/distributed-transaction/` - For registration workflows
- `patterns/entity/event-sourced/` - For attendee journey tracking
