# Tour Operations

Tour packaging, itinerary management, and guide coordination.

## NAICS Code

**561510** - Travel Agencies

## Scope

This model covers tour operator functions, including:

- Tour package design and pricing
- Itinerary planning and scheduling
- Guide and vehicle assignment
- Supplier and vendor coordination
- Traveler communication and documentation

## Key Concepts

| Concept | Description |
|---------|-------------|
| Tour | A packaged travel experience with defined itinerary |
| Itinerary | The sequence of activities, transfers, and accommodations |
| Booking | A customer reservation for a tour departure |
| Guide | A tour leader assigned to a departure |
