# Audit Management

Audit planning, testing, and reporting.

## Scope

This model covers audit engagement operations, including:

- Audit planning and risk assessment
- Control testing and sampling
- Substantive procedures
- Finding documentation
- Report preparation and issuance

## Key Concepts

| Concept | Description |
|---------|-------------|
| AuditPlan | Scope, timing, and procedures |
| Control | A testable internal control |
| Finding | A documented exception or deficiency |
| Opinion | The auditor's conclusion |
