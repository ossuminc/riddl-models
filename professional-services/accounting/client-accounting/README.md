# Client Accounting

Client engagement, workpapers, and service delivery.

## NAICS Code

**541211** - Offices of Certified Public Accountants

## Scope

This model covers client accounting operations, including:

- Engagement setup and planning
- Workpaper management
- Trial balance and adjustments
- Financial statement preparation
- Review and delivery

## Key Concepts

| Concept | Description |
|---------|-------------|
| Engagement | A client service agreement |
| Workpaper | Documentation supporting the work |
| TrialBalance | Account balances for review |
| Deliverable | A completed work product |
