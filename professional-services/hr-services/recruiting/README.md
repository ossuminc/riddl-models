# Recruiting

Job requisitions, candidate management, and hiring.

## Scope

This model covers recruiting operations, including:

- Requisition creation and approval
- Job posting and sourcing
- Candidate tracking and screening
- Interview scheduling and feedback
- Offer management and onboarding

## Key Concepts

| Concept | Description |
|---------|-------------|
| Requisition | An approved request to fill a position |
| Candidate | An applicant for a position |
| Interview | A scheduled evaluation session |
| Offer | Employment terms presented to a candidate |

- `patterns/saga/distributed-transaction/` - For hiring workflows
