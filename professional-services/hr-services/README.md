# HR Services

Human resources outsourcing and services.

## Scope

This subsector covers HR service provider operations, including:

- Payroll processing
- Benefits administration
- Recruiting and staffing
- HR compliance
- Employee self-service

## Models

| Model | Description |
|-------|-------------|
| `payroll-processing/` | Pay calculation, taxes, and disbursement |
| `benefits-administration/` | Enrollment, eligibility, and claims |
| `recruiting/` | Job postings, applicants, and hiring |

## Related Patterns

- `patterns/saga/distributed-transaction/` - For payroll workflows
- `patterns/entity/aggregate-root/` - For employee records
