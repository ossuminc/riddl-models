# Live Events

Concerts, theater, and live entertainment.

## Scope

This subsector covers live entertainment production, including:

- Event ticketing and seating
- Venue operations and logistics
- Production and stage management
- Artist and talent management
- Fan engagement and merchandise

## Models

| Model | Description |
|-------|-------------|
| `ticketing/` | Ticket sales, seating, and access control |
| `production-management/` | Show planning, staging, and crew |

## Related Patterns

- `patterns/entity/event-sourced/` - For ticket lifecycle
- `patterns/saga/distributed-transaction/` - For ticket purchases
