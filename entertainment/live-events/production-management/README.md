# Production Management

Show production, crew scheduling, and equipment coordination.

## NAICS Code

**711310** - Promoters of Performing Arts and Similar Events

## Scope

This model covers event production operations, including:

- Show planning and scheduling
- Crew assignment and availability
- Equipment inventory and allocation
- Venue technical requirements
- Run-of-show and cue management

## Key Concepts

| Concept | Description |
|---------|-------------|
| Production | A show or event being produced |
| Crew | Technical and production personnel |
| Equipment | Gear and assets required for production |
| RunOfShow | The timeline and cues for an event |
