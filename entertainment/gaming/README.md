# Gaming

Video games, online gaming, and virtual worlds.

## Scope

This subsector covers gaming platform and in-game operations, including:

- Player matchmaking and lobbies
- In-game economy and currencies
- Leaderboards and achievements
- Virtual goods and microtransactions
- Anti-cheat and moderation

## Models

| Model | Description |
|-------|-------------|
| `matchmaking/` | Player matching, lobbies, and session management |
| `game-economy/` | Virtual currencies, items, and trading |
