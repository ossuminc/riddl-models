# Sports

Professional and amateur sports organizations.

## Scope

This subsector covers sports organization management, including:

- League and team management
- Athlete contracts and performance
- Game scheduling and officiating
- Fan engagement and ticketing
- Statistics and analytics

## Models

| Model | Description |
|-------|-------------|
| `league-management/` | Teams, schedules, and standings |
| `athlete-management/` | Contracts, performance, and health |

## Related Patterns

- `patterns/entity/aggregate-root/` - For team/league hierarchies
- `patterns/projection/read-model/` - For statistics views
