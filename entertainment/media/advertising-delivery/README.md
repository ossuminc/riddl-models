# Advertising Delivery

Ad serving, targeting, and measurement for digital media.

## Scope

This model covers ad delivery operations, including:

- Ad inventory and placement
- Audience targeting and segments
- Real-time bidding integration
- Impression and click tracking
- Campaign measurement and attribution

## Key Concepts

| Concept | Description |
|---------|-------------|
| AdUnit | A placement for advertising content |
| Campaign | An advertiser's set of ads with targeting |
| Impression | A recorded ad view event |
| Targeting | Rules for audience selection |
