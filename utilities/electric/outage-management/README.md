# Outage Management

Power outage detection, dispatch, and restoration.

## NAICS Code

**221122** - Electric Power Distribution

## Scope

This model covers outage management operations, including:

- Outage detection and prediction
- Customer trouble reporting
- Crew dispatch and tracking
- Restoration prioritization
- Customer communication

## Key Concepts

| Concept | Description |
|---------|-------------|
| Outage | A power interruption event |
| Crew | Field personnel for restoration |
| TroubleCall | A customer-reported issue |
| Restoration | The process of returning power |
