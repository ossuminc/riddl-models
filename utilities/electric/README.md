# Electric Utilities

Electric power generation, transmission, and distribution.

## Scope

This subsector covers electric utility operations, including:

- Grid operations and dispatch
- Outage management
- Demand response
- Renewable integration
- SCADA and automation

## Models

| Model | Description |
|-------|-------------|
| `grid-operations/` | Generation dispatch and load balancing |
| `outage-management/` | Outage detection, crew dispatch, and restoration |

## Related Patterns

- `patterns/workflow/process-manager/` - For restoration workflows
- `patterns/entity/event-sourced/` - For outage history
