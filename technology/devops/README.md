# DevOps

Development operations, continuous delivery, and site reliability.

## Scope

This subsector covers the automation and tooling for software delivery and
operations, including:

- CI/CD pipeline orchestration
- Deployment automation and rollbacks
- Incident detection and response
- Infrastructure as code
- Monitoring and alerting

## Models

| Model | Description |
|-------|-------------|
| `deployment-pipeline/` | Build, test, and deploy automation |
| `incident-management/` | Alert routing, escalation, and resolution |
