# Platform

Core platform services for identity, APIs, and developer experience.

## Scope

This subsector covers foundational platform capabilities that other services
build upon, including:

- API gateway and management
- Identity and access management
- Developer portals and documentation
- Rate limiting and quotas
- Service discovery and routing

## Models

| Model | Description |
|-------|-------------|
| `api-management/` | API lifecycle, versioning, and developer access |
| `identity-management/` | Authentication, authorization, and SSO |
