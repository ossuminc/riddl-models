# Records Management

Evidence, reports, and document retention for public safety.

## Scope

This model covers public safety records, including:

- Incident and case reports
- Evidence chain of custody
- Digital asset management
- Records retention policies
- FOIA request processing

## Key Concepts

| Concept | Description |
|---------|-------------|
| Record | An official document with retention requirements |
| Evidence | Physical or digital items with chain of custody |
| RetentionPolicy | Rules governing record lifecycle |
| FOIARequest | A public records access request |
