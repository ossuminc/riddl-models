# Public Safety

Emergency services and public safety operations.

## Scope

This subsector covers public safety operations, including:

- Emergency dispatch (911/CAD)
- Records management
- Evidence and case tracking
- Resource deployment
- Interagency coordination

## Models

| Model | Description |
|-------|-------------|
| `emergency-dispatch/` | Call taking, dispatch, and unit tracking |
| `records-management/` | Incident reports, evidence, and cases |
