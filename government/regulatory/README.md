# Regulatory

Government regulatory and compliance functions.

## Scope

This subsector covers regulatory government operations, including:

- Licensing and registration
- Compliance reporting
- Inspections and audits
- Enforcement actions
- Regulatory filings

## Models

| Model | Description |
|-------|-------------|
| `licensing/` | License applications, renewals, and enforcement |
| `compliance-reporting/` | Filing requirements and submissions |
