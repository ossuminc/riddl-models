# Benefits Administration

Government benefits eligibility, enrollment, and payments.

## Scope

This model covers benefits programs, including:

- Program eligibility determination
- Application and enrollment
- Benefit calculation and issuance
- Recertification and renewals
- Overpayment recovery

## Key Concepts

| Concept | Description |
|---------|-------------|
| Program | A government benefit program with eligibility rules |
| Application | A citizen's request for benefits |
| Enrollment | Active participation in a benefits program |
| Payment | A benefit disbursement to a recipient |

## Related Patterns

- `patterns/saga/distributed-transaction/` - For enrollment workflows
- `patterns/entity/event-sourced/` - For eligibility history
