# Case Management

Citizen case intake, investigation, and resolution.

## NAICS Code

**921190** - Other General Government Support

## Scope

This model covers government case management, including:

- Case intake and assignment
- Investigation and documentation
- Eligibility determination
- Decision and appeals
- Case closure and outcomes

## Key Concepts

| Concept | Description |
|---------|-------------|
| Case | A citizen matter requiring investigation or decision |
| Caseworker | Staff assigned to manage cases |
| Document | Evidence or correspondence attached to a case |
| Decision | The outcome determination for a case |
