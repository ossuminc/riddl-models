# Citizen Services

Government services for citizens and residents.

## Scope

This subsector covers citizen-facing government services, including:

- Benefits administration
- Case management
- Permit and license applications
- Vital records
- Public assistance programs

## Models

| Model | Description |
|-------|-------------|
| `benefits-administration/` | Eligibility, enrollment, and payments |
| `case-management/` | Intake, investigation, and resolution |
| `permit-management/` | Application, review, and issuance |

## Related Patterns

- `patterns/workflow/process-manager/` - For application workflows
- `patterns/entity/event-sourced/` - For case history
