# Timber Management

Forest timber operations, harvest planning, and regeneration.

## NAICS Code

**113310** - Logging

## Scope

This model covers timber management operations, including:

- Timber stand registration and inventory
- Cruise data recording and volume estimates
- Harvest planning and timber sales
- Logging operations and load tracking
- Stand regeneration and planting
- Growth monitoring and stand updates

## Key Concepts

| Concept | Description |
|---------|-------------|
| Stand | A defined forest area with timber inventory |
| CruiseData | Timber volume estimates from field surveys |
| TimberSaleInfo | Sale details with buyer, price, and method |
| HarvestLoad | Individual load of harvested timber |
| RegenerationPlan | Post-harvest reforestation plan |
