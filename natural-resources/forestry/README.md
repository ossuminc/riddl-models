# Forestry

Timber harvesting and forest management.

## Scope

This subsector covers forestry operations, including:

- Forest inventory and planning
- Harvest scheduling
- Log tracking and scaling
- Reforestation and silviculture
- Environmental compliance

## Models

| Model | Description |
|-------|-------------|
| `harvest-planning/` | Cut blocks, permits, and scheduling |
| `log-tracking/` | Log scaling, transport, and custody |
