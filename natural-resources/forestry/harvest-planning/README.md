# Harvest Planning

Timber inventory, cutting plans, and harvest scheduling.

## NAICS Code

**113310** - Logging

## Scope

This model covers harvest planning operations, including:

- Forest inventory and cruising
- Cutting block delineation
- Harvest scheduling and permits
- Road planning and access
- Silviculture and reforestation

## Key Concepts

| Concept | Description |
|---------|-------------|
| Stand | A defined forest area with timber |
| CuttingBlock | An approved area for harvest |
| CruiseData | Timber volume estimates |
| HarvestPlan | Scheduled extraction activities |
