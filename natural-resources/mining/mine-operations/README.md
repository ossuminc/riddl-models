# Mine Operations

Mine planning, extraction, and processing operations.

## NAICS Code

**212000** - Mining (except Oil and Gas)

## Scope

This model covers mining operations, including:

- Mine planning and scheduling
- Extraction and haulage
- Processing and beneficiation
- Equipment and fleet management
- Safety and environmental compliance

## Key Concepts

| Concept | Description |
|---------|-------------|
| Mine | A mineral extraction site |
| Block | A defined area with estimated ore |
| Extraction | Removal of material from ground |
| Processing | Separation and concentration of ore |
