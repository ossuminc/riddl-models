# Mining

Mining operations and mineral extraction.

## Scope

This subsector covers mining and mineral processing, including:

- Mine planning and operations
- Ore tracking and grade control
- Equipment fleet management
- Safety and compliance
- Environmental monitoring

## Models

| Model | Description |
|-------|-------------|
| `mine-operations/` | Extraction, haulage, and processing |
| `mineral-tracking/` | Ore grades, stockpiles, and reconciliation |
