# Property Management

Lease administration, tenant services, and maintenance.

## Scope

This model covers property management operations, including:

- Lease creation and renewals
- Rent collection and accounting
- Tenant communication and requests
- Maintenance work orders
- Vendor coordination

## Key Concepts

| Concept | Description |
|---------|-------------|
| Property | A managed real estate asset |
| Lease | A tenant occupancy agreement |
| Tenant | An occupant with lease obligations |
| WorkOrder | A maintenance or repair request |

## Related Patterns

- `patterns/entity/event-sourced/` - For lease history
- `patterns/workflow/process-manager/` - For maintenance coordination
