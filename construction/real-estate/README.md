# Real Estate

Property management and real estate transactions.

## Scope

This subsector covers commercial and residential real estate, including:

- Property listing and marketing
- Transaction management
- Lease administration
- Property maintenance
- Tenant relations

## Models

| Model | Description |
|-------|-------------|
| `property-management/` | Buildings, units, and maintenance |
| `transaction-management/` | Listings, offers, and closings |

## Related Patterns

- `patterns/entity/aggregate-root/` - For property hierarchies
- `patterns/saga/distributed-transaction/` - For transaction workflows
