# Construction Project

Project planning, scheduling, and cost control.

## NAICS Code

**236200** - Nonresidential Building Construction

## Scope

This model covers construction project management, including:

- Project setup and budgeting
- Schedule management (CPM)
- Cost tracking and forecasting
- Change order management
- Progress reporting

## Key Concepts

| Concept | Description |
|---------|-------------|
| Project | A construction endeavor with scope and budget |
| Schedule | Critical path timeline of activities |
| Budget | Allocated costs by category |
| ChangeOrder | An approved scope modification |
