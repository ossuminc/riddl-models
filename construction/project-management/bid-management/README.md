# Bid Management

RFP response, estimating, and contract awards.

## Scope

This model covers bid management operations, including:

- Opportunity tracking and qualification
- Estimate preparation and review
- Bid submission and tracking
- Award notification and contracting
- Post-bid analysis

## Key Concepts

| Concept | Description |
|---------|-------------|
| Opportunity | A potential project to bid |
| Estimate | Cost and resource projection |
| Bid | A formal proposal submission |
| Award | Selection for contract |
