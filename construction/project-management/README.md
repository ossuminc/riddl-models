# Project Management

Construction project planning and execution.

## Scope

This subsector covers construction project management, including:

- Bid preparation and submission
- Project scheduling and milestones
- Subcontractor coordination
- Budget and cost tracking
- Change order management

## Models

| Model | Description |
|-------|-------------|
| `construction-project/` | Schedule, budget, and milestone tracking |
| `bid-management/` | Estimating, proposal, and submission |
| `subcontractor-management/` | Contracts, performance, and payments |

- `patterns/entity/aggregate-root/` - For project hierarchies
