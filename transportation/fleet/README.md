# Fleet

Vehicle fleet management and optimization.

## Scope

This subsector covers commercial fleet operations, including:

- Vehicle acquisition and disposal
- Maintenance scheduling
- Driver management
- Fuel management
- Route optimization

## Models

| Model | Description |
|-------|-------------|
| `fleet-management/` | Vehicles, drivers, and compliance |
| `route-optimization/` | Routing, scheduling, and dispatch |

## Related Patterns

- `patterns/entity/aggregate-root/` - For fleet hierarchies
- `patterns/entity/event-sourced/` - For vehicle history
