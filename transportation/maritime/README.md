# Maritime

Shipping, port operations, and vessel management.

## Scope

This subsector covers maritime transportation, including:

- Vessel scheduling and operations
- Port and terminal operations
- Container tracking
- Cargo stowage planning
- Maritime safety and compliance

## Models

| Model | Description |
|-------|-------------|
| `vessel-management/` | Fleet, crew, and voyage planning |
| `port-operations/` | Berth scheduling, cargo handling, and yard |
