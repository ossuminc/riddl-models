# Credentialing

Professional certification exams, issuance, and renewals.

## NAICS Code

**611710** - Educational Support Services

## Scope

This model covers credentialing operations, including:

- Certification program management
- Exam registration and administration
- Credential issuance and verification
- Continuing education tracking
- Renewal and recertification

## Key Concepts

| Concept | Description |
|---------|-------------|
| Certification | A professional credential with requirements |
| Exam | An assessment for credential qualification |
| Credential | An issued certificate with validity period |
| Renewal | The process of maintaining certification |
