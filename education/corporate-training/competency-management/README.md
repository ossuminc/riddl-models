# Competency Management

Skills tracking, assessments, and development planning.

## NAICS Code

**611430** - Professional and Management Development Training

## Scope

This model covers competency management, including:

- Skill and competency frameworks
- Employee skill assessments
- Gap analysis and recommendations
- Development plan creation
- Competency verification and certification

## Key Concepts

| Concept | Description |
|---------|-------------|
| Competency | A defined skill or capability |
| Assessment | An evaluation of competency level |
| SkillGap | Difference between required and current skills |
| DevelopmentPlan | A path to acquire needed competencies |
