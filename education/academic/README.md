# Academic

Higher education and K-12 academic operations.

## Scope

This subsector covers academic institution operations, including:

- Student information systems
- Course catalog and scheduling
- Learning management
- Grading and transcripts
- Academic advising

## Models

| Model | Description |
|-------|-------------|
| `student-information/` | Enrollment, demographics, and records |
| `course-management/` | Catalog, sections, and scheduling |
| `learning-management/` | Content delivery, assignments, and grades |

## Related Patterns

- `patterns/entity/aggregate-root/` - For student records
- `patterns/entity/event-sourced/` - For academic history
