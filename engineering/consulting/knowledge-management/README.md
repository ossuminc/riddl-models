# Knowledge Management

Expertise directory, lessons learned, and knowledge sharing.

## NAICS Code

**541330** - Engineering Services

## Scope

This model covers knowledge management, including:

- Expert directory and skills inventory
- Lessons learned capture and search
- Best practices documentation
- Knowledge article management
- Communities of practice

## Key Concepts

| Concept | Description |
|---------|-------------|
| Expert | A person with documented expertise |
| LessonLearned | An insight from project experience |
| Article | A knowledge base document |
| Community | A group sharing domain knowledge |
