# Engagement Management

SOW management, resource allocation, and client billing.

## Scope

This model covers consulting engagement operations, including:

- Statement of work creation
- Resource planning and assignment
- Time and expense tracking
- Milestone and deliverable management
- Invoicing and collections

## Key Concepts

| Concept | Description |
|---------|-------------|
| Engagement | A client consulting project |
| SOW | Statement of Work defining scope |
| Resource | A consultant assigned to work |
| Invoice | Billing for services rendered |

## Related Patterns

- `patterns/entity/event-sourced/` - For engagement history
- `patterns/saga/distributed-transaction/` - For billing workflows
