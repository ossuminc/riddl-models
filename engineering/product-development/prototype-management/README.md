# Prototype Management

Prototype builds, testing, and iteration tracking.

## Scope

This model covers prototype management, including:

- Build request and planning
- Parts procurement and assembly
- Test plan execution
- Issue tracking and resolution
- Iteration and version control

## Key Concepts

| Concept | Description |
|---------|-------------|
| Prototype | A test version of a product |
| Build | An assembly of a prototype version |
| TestPlan | Procedures to validate the design |
| Issue | A problem discovered during testing |
